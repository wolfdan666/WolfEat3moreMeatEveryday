# 对象模型:关于vptr和vtbl
一个虚函数和一百个虚函数都是一个虚函数指针

**继承是继承父类的调用权**

vptr(virtual pointer)指向虚表vtbl(virtual table)



```cpp
class A{
public:
    virtual void vfunc1();
    virtual void vfunc2();
    void func1();
    void func2();
private:
    int m_data1,m_data2;
};

class B: public A{
public:
    // 定义自己的vfunc1(),重载掉之前的
    virtual void vfunc1();
    void func2();
private:
    int m_data3;
};

class C: public B{
public:
    virtual void vfunc1();
    void func2();
private:
    int m_data1,m_data4;
};
```

## 关于list一串不同的对象
由于list要存储相同大小的存储空间，所以直接用指针，
然后指针指向不同的对象

然后根据指针指向的对象直接调用虚函数draw(),根据不同的对象

## 动态绑定条件
- 动态绑定必须是指针调用
- 必须是upcast，子类可以动态调用父类的函数
- virtual function

编译成下面这样子
```cpp
(*(p->vptr)[n])(p);
// or 上下是一样的
(* p->vptr[n] )(p);
```

最后的`(p)`是表示this指针
